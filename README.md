# MCU Doc AI — Releases

Gwanak Analog MCU 개발자 지원 앱의 **배포 채널** 입니다.

## 📦 다운로드

최신 인스톨러는 [Releases](https://github.com/lg01025484033-commits/mcu-doc-ai-releases/releases/latest) 페이지에서 받을 수 있습니다.

## ⚙️ 자동 업데이트

이 저장소는 `electron-updater` 의 업데이트 소스로 사용됩니다. 설치된 앱이 6시간마다 자동으로 새 버전을 감지합니다.

## 🔗 설치 방법

1. [Releases](https://github.com/lg01025484033-commits/mcu-doc-ai-releases/releases) 페이지에서 최신 `MCU Doc AI Setup X.X.X.exe` 다운로드
2. 실행 후 설치 마법사 진행
3. 최초 실행 시 라이선스 키 입력 (담당 FAE 에게 문의)
